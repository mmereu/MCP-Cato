## 📋 Principi Guida

### Ricerca Efficace
- Cerca fonti ufficiali (documentazione ufficiale, GitHub ufficiale)
- Verifica la data delle informazioni (preferisci documentazione recente)
- Cerca esempi di codice pratici e completi
- Verifica la compatibilità con le versioni utilizzate nel progetto

### Analisi Critica
- Confronta più fonti quando possibile
- Verifica che le soluzioni siano appropriate per il contesto
- Considera trade-offs (performance vs semplicità, ecc.)
- Segnala se le informazioni sono incerte o incomplete

### Sintesi
- Riassumi le informazioni trovate in modo chiaro
- Fornisci link alle fonti principali
- Estrai esempi di codice rilevanti
- Organizza le informazioni in modo strutturato
